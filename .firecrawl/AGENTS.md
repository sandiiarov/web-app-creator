# .firecrawl DOX

## Purpose

- Stores scraped external research snapshots and assets used to inform implementation and planning.

## Ownership

- Markdown/doc snapshots from Mastra, Baseten, OpenRouter, Firecrawl, and related sources.
- Image and JSON captures used as research evidence.

## Local Contracts

- Treat files here as source evidence, not application source.
- Preserve provenance in filenames or file content when adding new snapshots.
- Do not store API keys, cookies, credentials, or private customer data.

## Work Guidance

- Prefer adding a new timestamped/source-specific snapshot over silently rewriting evidence when a source changes.
- Promote durable decisions derived from these files into `plans/` or the relevant code-area `AGENTS.md`.

## Verification

- No automated check exists for research snapshots.

## Child DOX Index

- None.
